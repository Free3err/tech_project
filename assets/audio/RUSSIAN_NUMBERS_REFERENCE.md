## Usage in Code / Использование в коде

The `generate_audio_files.py` script uses this mapping:

```python
RUSSIAN_NUMBERS = {
    1: 'один', 2: 'два', 3: 'три', 4: 'четыре', 5: 'пять',
    6: 'шесть', 7: 'семь', 8: 'восемь', 9: 'девять', 10: 'десять',
    11: 'одиннадцать', 12: 'двенадцать', 13: 'тринадцать',
    14: 'четырнадцать', 15: 'пятнадцать', 16: 'шестнадцать',
    17: 'семнадцать', 18: 'восемнадцать', 19: 'девятнадцать',
    20: 'двадцать', 30: 'тридцать', 40: 'сорок', 50: 'пятьдесят',
    60: 'шестьдесят', 70: 'семьдесят', 80: 'восемьдесят', 90: 'девяносто',
    100: 'сто'
}

def number_to_russian(n):
    if n in RUSSIAN_NUMBERS:
        return RUSSIAN_NUMBERS[n]
    elif n < 100:
        tens = (n // 10) * 10
        ones = n % 10
        return f"{RUSSIAN_NUMBERS[tens]} {RUSSIAN_NUMBERS[ones]}"
    return str(n)
```

## Pronunciation Tips / Советы по произношению

For native English speakers recording Russian numbers:

1. **Soft consonants**: Many Russian consonants are "softened" - indicated by ' in transliteration
2. **Stress**: Russian words have one stressed syllable (shown in CAPS in pronunciation guide)
3. **Vowel reduction**: Unstressed vowels are often reduced/shortened
4. **Hard 'r'**: Russian 'р' is rolled/trilled
5. **'ь' (soft sign)**: Makes preceding consonant soft (like adding a slight 'y' sound)
